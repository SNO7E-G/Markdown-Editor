Markdown Cheat Sheet

Basic Syntax

  • # Heading 1
  • ## Heading 2
  • **Bold**
  • *Italic*
  • ~~Strikethrough~~

Lists

  • - Unordered
  • 1. Ordered
  • - [ ] Task
  • - [x] Done

Links & Images

  • [Link](url)
  • ![Image](url)
  • [Link][ref]
  • [ref]: url

Code

  • `Inline`
  • ```Block```
  • ```js
  • Indented

Shortcuts

  • Ctrl+B: Bold
  • Ctrl+I: Italic
  • Ctrl+K: Link
  • Ctrl+Shift+K: Image

Other

  • > Quote
  • ---
  • |Table|Header|
  • [^1]: Footnote
